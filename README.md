# TaxPrep for Etsy

iOS application to help Etsy sellers organize transaction data for Schedule C tax filing.

## Status
Currently in development. Awaiting Etsy API approval.

## Features (Planned)
- Read-only access to Etsy shop data
- Transaction history retrieval
- Schedule C income categorization
- Local data storage (no third-party servers)

## Privacy
All data stored locally on device using iOS secure storage.
